CLASSiFIND

Android application for scheduling study time in open classrooms.

At Johns Hopkins University, there are multiple buildings where students may use the space to study when class is not in session. CLASSiFIND pulls data from the JHU course search API to populate a calendar with the times that a room is occupied for a class. Then, CLASSiFIND allows the user to schedule up to 2 hours in a room. Other users will see this time in the calendar and will therefore know when another student plans to be in a specific classroom. The app also features a map that will direct users to the location of the building that they scheduled time at.
