Memory Augmented Conversational Language Models

Research on using a two-model pipeline to improve the recall of Large Language Models.

The two-model pipeline consists of a fine-tuned a BART-based “unlimiformer” model for dialogue summarization and a chatbot that can utilize the summary as context to extend its context window.

Technology

Machine Learning in Python (PyTorch and scikit-learn)
